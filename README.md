## 📖 О проекте

**Lobby** — это образовательная платформа с элементами геймификации, разработанная для детского технопарка **«Кванториум»**. Проект сочетает в себе Telegram Mini App и Telegram-бота, превращая учебный процесс в увлекательную игру с достижениями, баллами и прогрессом, при этом не добавлябщая работы педагогам за счет парсинга данных из их учетных электронных таблиц

> 💡 Проект создан в образовательных целях и представляет собой прототип платформы для геймификации обучения в формате технопарка.

## ✨ Ключевые идеи

| Функция | Описание |
|--------|----------|
| 🎯 **Геймификация** | Превращение учёбы в игру с очками, уровнями и наградами |
| 🤖 **Telegram-бот** | Интерактивный бот для взаимодействия с платформой |
| 📱 **Mini App** | Веб-интерфейс прямо внутри Telegram |
| 🗄️ **База данных** | Хранение прогресса и достижений пользователей |

## 🛠️ Технологии

```
Frontend  │  HTML · CSS · JavaScript
Backend   │  Pytgon (FastApi)
Database  │  SQL (реляционная БД)
Platform  │  Telegram Bot API · Telegram Mini Apps
```

**Стек по языкам:**
- 🐍 **Python** — 53% — серверная логика и Telegram-бот
- ⚡ **JavaScript** — 28% — интерактивность Mini App
- 🎨 **CSS** — 15% — стили и оформление
- 🌐 **HTML** — 5% — разметка интерфейса
